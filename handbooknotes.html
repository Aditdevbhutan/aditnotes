# Python Quick Reference Guide

## Basics

### Quotes
Use quotes when you want text and leave them out if you want to compute something. 
```python
print("2+2")  # Prints: 2+2
print(2+2)  # Prints: 4

Getting Input

Use eval with input when getting a number, input alone when getting text.

name = input("Enter your name: ")
num = eval(input("Enter a number: "))

Print Function

Optional Arguments:

  • sep - Change or remove spaces that print inserts
  • end - Change or remove when print jumps to the next line
print(1, 2, 3, 4, 5, sep='')  # prints 12345
print(1, 2, 3, 4, 5, sep='/')  # prints 1/2/3/4/5

for i in range(10):
    print(i, end="")  # Prints: 0123456789

Math Operators

Operator Meaning Example
** Power 5**2 = 25
% Modulo (Remainder) 19%5 = 4
// Integer Division 5//3 = 1 
from math import sin, pi
print("The sine of pi is", sin(pi))
print("The absolute value of -3 is", abs(-3))
print("4/3 rounded to two decimal places is", round(4/3, 2))

Random Numbers

from random import randint
x = randint(1, 5)  # Random number between 1 and 5

If Statements

if x == 3:  # if x is 3
if x != 3:  # if x is not 3
if x >= 1 and x <= 5:  # if x is between 1 and 5
if name == "elvis":  # checking strings

If/Else:

if guess == num:
    print("You got it!")
else:
    print("You missed it.")

If/Elif/Else:

if entry == "yes":
    print("Your total bill is $50.00")
elif entry == "no":
    print("Your total bill is $45.00")

Strings

s = "hello"
print(s.upper())  # HELLO
print(s.replace("h", "H"))  # Hello
print(len(s))  # 5

String Slicing

s = "abcdefghij"
print(s[:3])  # abc
print(s[-3:])  # hij
print(s[::-1])  # Reverse string

Common Programming Techniques

Counting

count = 0
for i in range(10):
    num = eval(input("Enter a number: "))
    if num == 7:
        count += 1
print(count)

Summing

total = 0
for i in range(1, 101):
    total += i
print("1+2+...+100 is", total)

Finding Max

longest_word = ""
for word in L:
    if len(word) > len(longest_word):
        longest_word = word
